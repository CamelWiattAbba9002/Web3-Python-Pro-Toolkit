# Web3 Python Pro Toolkit
区块链Python高级自动化工具集 | 16个全新独立脚本 | 支持ETH/BSC/NFT/DeFi/IPFS

## 项目介绍
本仓库是一套**纯Python开发**的专业级Web3区块链工具库，包含16个完全独立、无任何重复的高效脚本。覆盖批量转账、NFT持仓快照、多链监控、私钥安全、IPFS存储、数据分析、自动化任务等企业级场景，代码简洁、注释完整、可直接运行、可二次开发、可部署服务器。

专为区块链开发者、安全工程师、自动化运维、数据分析人员打造。

## 包含16个核心工具
1. batch_eth_transfer.py - 以太坊ETH批量转账工具
2. nft_holder_snapshot.py - NFT持有者地址快照统计
3. bsc_wallet_generator.py - BSC币安链钱包批量生成
4. tx_history_exporter.py - 钱包交易历史CSV导出
5. erc20_airdrop_tool.py - ERC20代币批量空投脚本
6. chain_id_checker.py - 全EVM公链ChainID一键查询
7. privatekey_encrypt_safe.py - 私钥AES加密安全存储
8. defi_pool_monitor.py - DeFi流动性池实时价格监控
9. wallet_balance_rich_list.py - 多地址资产余额排行榜
10. ipfs_file_uploader.py - 去中心化IPFS文件上传工具
11. abi_json_formatter.py - 智能合约ABI格式化美化
12. evm_chain_balance_checker.py - 多公链资产一键查询
13. contract_deploy_simulator.py - 合约部署Gas成本模拟
14. web3_auto_claimer.py - 链上奖励自动领取脚本
15. block_data_analyzer.py - 以太坊区块数据分析工具
16. token_security_checker.py - 代币安全风险检测工具
